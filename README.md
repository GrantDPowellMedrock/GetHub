# GetHub — a free, fast Git GUI client

[![license](https://img.shields.io/github/license/GrantDPowellMedrock/GetHub.svg)](LICENSE)
[![latest](https://img.shields.io/github/v/release/GrantDPowellMedrock/GetHub.svg)](https://github.com/GrantDPowellMedrock/GetHub/releases/latest)

GetHub is a fork of [SourceGit](https://github.com/sourcegit-scm/sourcegit) (MIT-licensed). SourceGit is an excellent, fast, cross-platform Git GUI — full credit to its authors.

This fork exists to layer on workflow-focused enhancements for people juggling a large number of repositories. Changes so far:

- **Group tabs** — repo folders surface as a tab strip; click a group to open all its repos
- **Color-coded groups** — right-click a group tab to assign a color
- **Drag-to-reorder** groups
- **Per-tab ahead/behind indicators** so you can see at a glance what needs a push/pull
- **"Open in Zed"** one-click toolbar shortcut
- **Version in the title bar** and a streamlined About dialog
- **Portable single-file build** (~26 MB, no .NET install required)

More improvements are ongoing. Upstream SourceGit changes are merged in periodically.
